# Utility Macros

## Thornmarch

You're _welcome_.

```
/tenemy "Whiskerwall Kupdi Koop"
/marking ignore1

/tenemy "Pukla Puki the Pomburner"
/marking attack1

/tenemy "WoolyWart Kupu Kogi"
/marking attack5

/tenemy "Furryfoot Kupli Kipp"
/marking attack2

/tenemy "Pukna Pako the Tailturner"
/marking attack4

/tenemy "Ruffletuft Kupta Kapa"
/marking ignore2

/tenemy "Puksi Piko the Shaggysong"
/marking attack3

```

Theoretically same thing, but fewer commands.

```
/enemysign ignore1 "Whiskerwall Kupdi Koop"
/enemysign attack1 "Pukla Puki the Pomburner"
/enemysign attack5 "WoolyWart Kupu Kogi"
/enemysign attack2 "Furryfoot Kupli Kipp"
/enemysign attack4 "Pukna Pako the Tailturner"
/enemysign ignore2 "Ruffletuft Kupta Kapa"
/enemysign attack3 "Puksi Piko the Shaggysong"
```
